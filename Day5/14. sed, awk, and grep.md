# 14. sed, awk, and grep — Practical One-liners

These tools are **text processing utilities** used in shell scripting for filtering, transforming, and analyzing text.

---

## 14.1 grep
- Searches for patterns in files or input

### Syntax
```bash
grep [options] pattern [file]
```

### Common Options
- `-i` → ignore case
- `-v` → invert match
- `-r` → recursive
- `-n` → show line numbers

### Examples
```bash
grep "error" /var/log/syslog             # Lines containing 'error'
grep -i "warning" file.txt               # Case-insensitive search
grep -rn "TODO" ~/projects/             # Search recursively with line numbers
```

---

## 14.2 sed (Stream Editor)
- Edit text in a **stream or file**

### Syntax
```bash
sed [options] 'command' file
```

### Examples
1. Replace text:
```bash
sed 's/old/new/g' file.txt   # Replace all 'old' with 'new'
```
2. Delete line:
```bash
sed '2d' file.txt             # Delete second line
```
3. Print specific line:
```bash
sed -n '3p' file.txt          # Print third line
```

---

## 14.3 awk
- Pattern scanning and **processing language**

### Syntax
```bash
awk 'pattern { action }' file
```

### Examples
1. Print column 1 and 3:
```bash
awk '{print $1, $3}' file.txt
```
2. Sum numbers in column 2:
```bash
awk '{sum += $2} END {print sum}' file.txt
```
3. Conditional print:
```bash
awk '$2 > 50 {print $1, $2}' file.txt
```

---

## 14.4 Practical One-liners
1. Find all `.log` files containing 'error' and show line numbers:
```bash
grep -rn "error" *.log
```
2. Replace `foo` with `bar` in-place:
```bash
sed -i 's/foo/bar/g' file.txt
```
3. Print usernames from `/etc/passwd`:
```bash
awk -F: '{print $1}' /etc/passwd
```
4. Count occurrences of a word:
```bash
grep -o "Linux" file.txt | wc -l
```
5. Extract second and third columns of CSV:
```bash
awk -F, '{print $2, $3}' data.csv
```
